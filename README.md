# HungrySam
為了將美食的喜悅分享給世界，Sam 創立了 HungrySam 這個美食部落客。

## 特點
- **Google 認證登入**: 提供用戶一個安全且快速的登入方式，透過 Google 帳戶直接認證。
- **內容互動**: 用戶可以自由地新增貼文，並對現有貼文進行回應、編輯及刪除。

## 前端技術
- **Bootstrap**
- **jQuery**
- **SCSS**

## 後端技術
- **EJS**
- **Node.js**
- **Express**

## 快速啟動
git clone https://github.com/Feederboyz/HungrySam.git
npm install
node start
